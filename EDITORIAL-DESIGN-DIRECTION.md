Design Direction Lock

We are designing the Editorial landing template for premium fashion brands.

Use this hybrid reference strategy:

Primary reference:
- Reformation-style structure and editorial ecommerce rhythm

Secondary reference:
- Quiet luxury mood similar to the warm, restrained "AESTHETIC" reference

Tertiary influence:
- Campaign storytelling rhythm similar to the Zanerobe-style layout, but only lightly

Target personality:
- premium fashion editorial
- quiet luxury
- image-led
- refined
- spacious
- clean
- ecommerce-ready
- storytelling-oriented

Avoid:
- streetwear aggression
- overly busy utility bars
- generic Shopify retail feeling
- loud typography gimmicks
- too many small UI elements
- SaaS-like spacing

System constraints:
- preserve existing CMS-driven header system
- preserve existing layout settings
- preserve logo/text/logo-size/navigation source from CMS
- no hardcoded brand values
- no architecture rewrite
