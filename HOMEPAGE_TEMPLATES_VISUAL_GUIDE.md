## 🎯 Template Selection Guide

### Choose **Modern** if you want:

- ✓ Contemporary web design trends
- ✓ Tech-forward aesthetic
- ✓ Gradient backgrounds and depth
- ✓ Professional but approachable

### Choose **Classic** if you want:

- ✓ Traditional e-commerce layout
- ✓ Trust badges and conversion focus
- ✓ Clean, efficient space usage
- ✓ Familiar shopping experience

### Choose **Editorial** if you want:

- ✓ Luxury brand positioning
- ✓ Magazine-style storytelling
- ✓ Dramatic full-screen sections
- ✓ Premium, high-end aesthetic

### Choose **Minimal** if you want:

- ✓ Typography as hero element
- ✓ Clean, distraction-free experience
- ✓ Modern professional vibe
- ✓ Maximum whitespace and breathing room

## 📐 Responsive Behavior

All templates are **fully responsive** with these breakpoints:

```
Mobile First Approach:
- Base:     < 640px   (mobile)
- sm:       640px     (large mobile)
- md:       768px     (tablet)
- lg:       1024px    (desktop)
- xl:       1280px    (large desktop)
```

### Typography Scaling Example (Hero):

```
Modern:     text-4xl sm:text-5xl lg:text-6xl
Classic:    text-4xl sm:text-5xl lg:text-6xl
Editorial:  text-6xl sm:text-7xl lg:text-8xl
Minimal:    text-5xl sm:text-6xl lg:text-7xl xl:text-8xl
```

### Layout Adaptations:

- **Grid columns:** 1 col mobile → 2-4 cols desktop
- **Padding:** Reduced on mobile (py-12 → py-20 → py-28)
- **Typography:** Scaled down on mobile (text-4xl → text-8xl)
- **Spacing:** Compact on mobile, generous on desktop

## ✨ Usage Example

```tsx
import {
  LandingTemplateModern,
  LandingTemplateClassic,
  LandingTemplateEditorial,
  LandingTemplateMinimal
} from '#root/components/template-system';

// Same content powers all 4 templates!
const content: HomepageContent = {
  hero: {
    enabled: true,
    title: "Transform Your Shopping Experience",
    subtitle: "Discover curated products from trusted vendors",
    ctaText: "Shop Now",
    ctaLink: "/shop",
    imageUrl: "/hero.jpg"
  },
  // ... other sections
};

// Render any template
<LandingTemplateModern content={content} featuredProducts={products} />
<LandingTemplateClassic content={content} featuredProducts={products} />
<LandingTemplateEditorial content={content} featuredProducts={products} />
<LandingTemplateMinimal content={content} featuredProducts={products} />
```
